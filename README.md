# 学生信息管理系统（简易版） python700

## 项目概述

学生信息管理系统（简易版）是基于Python语言开发的一个小型的信息管理工具，旨在帮助学校或教育机构高效管理学生数据。

## 技术栈

- 后端：Python
- 数据存储：内置数据结构（如列表、字典）

## 功能模块

### 基础操作

- 录入学生信息
- 查找学生信息
- 删除学生信息
- 修改学生信息
- 排序学生信息
- 统计学生总人数
- 显示学生信息

### 系统退出

- 提供退出系统的功能

## 系统角色

- 管理员：负责所有对学生信息的增删改查操作

## 运行环境

- Python环境
- 推荐使用Python 3.x版本

## 目录结构

```
student_info_system/
```
